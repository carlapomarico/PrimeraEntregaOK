Welcome

This site was buit as a project for Codo a CodoFullstack Python Web Development Course, in Buenos Aires, March to July 2022.

What must be submitted?

The OPT must meet the following requirements:

  • Having at least 3 html pages, being its subject matter free. In case it is a One Page style Website, it must have at least 3 sections.
  • Using semantic tags to structure the site.
  • Containing a contact form with validation through Javascript.
  • Using at least one Iframe, FontAwesome icons, and local fonts or Google Fonts.
  • Having at least one fully responsive page with 3 breakpoints for 3 different device sizes.
  • Using at least one animation, transformation or transition (indicate where the animation was incorporated).
  • Having an HTML structure laid out with Flexbox and/or Grid.
  • If you want to use Bootstrap (not required), you may only do so on the form page of the website.
  • The page must be uploaded to an on-line server to be browsed by the Teacher. Example: GitHub Pages, Netlify, 000webhost or similar.
  • In addition to the technical aspects already mentioned, the project should include design aspects which will be evaluated by the teacher. For that purpose, it is recommended that you view the material available in the Virtual Classroom. Web Design Seminar. For more information, see material included in the Virtual Classroom Methodologies and tools I/Design.

Note: the OPT is individual but in case you want to solve the project as a group, you must synchronize the code of the OPT in Github (or similar) sending the link of the repository to the teacher. This mandatory requirement only applies to projects developed by 2 people (maximum number of members possible). For more information about Git, see material included in the Virtual Classroom Methodologies and tools I/Git.

Passing conditions: to pass, the practical work must be solved in a 60% correct way.
